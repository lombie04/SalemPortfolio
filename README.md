# Salem Lombard — Recruiter-Friendly Personal Website

## Quick start
- Open `index.html` in your browser.
- Deploy by uploading the folder to any static host (GitHub Pages / Netlify / Render static / etc.).

## What to edit first (10 minutes)
1. **Links**
   - In `index.html` search for `update link` and replace placeholders:
     - LinkedIn
     - GitHub
     - Resume PDF link
2. **Email**
   - In `index.html`, replace `your.email@example.com` in the contact form `mailto:`.
3. **Projects**
   - Edit the `PROJECTS` array in `assets/js/main.js`.
   - Keep 3–6 strong projects.

## Optional improvements
- Add a PDF CV: put it in `assets/Salem_Lombard_CV.pdf` and update the Resume link in `index.html`.
- Replace the avatar initials with a photo: add an image to `assets/img/` and swap the avatar div for an `<img>`.

## Legacy pages
Your previous multi-page site is kept in `/legacy/`.
